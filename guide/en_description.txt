## üîí Privacy & Security

**100% Local Storage**
All your profiles and data are stored exclusively on your device using the browser's secure storage API. No cloud servers, no data mining, no tracking. Your information never leaves your browser.

**Security Audited (v1.0.6)**
This release includes a comprehensive internal security audit. Key findings and measures:
- No `eval()` or `new Function()` usage anywhere in the codebase
- No external `fetch()` calls ‚Äî all network requests are extension-internal
- All user-controlled strings rendered through `innerHTML` are escaped with `escapeHtml()`
- `chrome.runtime.onMessage` validates `sender.id === chrome.runtime.id` in all listeners
- `postMessage` iframe relay validates `event.source` against a known frame hierarchy
- JSON import capped at 5 MB with structural validation before processing
- Notification messages use `textContent` (never `innerHTML`) for user-facing text

**Minimal Permissions**
We only request permissions essential for functionality:
- Storage: Save profiles locally
- ActiveTab: Interact with current forms
- Scripting: Inject toolbar and fill functionality
- Clipboard: Copy/paste profile data
- ContextMenus: Quick access shortcuts
- DeclarativeNetRequest: Modify network requests for user agent switching
- DeclarativeNetRequestWithHostAccess: Extended network request handling
- Downloads: Save HTML exports and screenshots
- WebNavigation: Detect URL changes in Single-Page Apps (SPA) to auto-refresh the profile list without requiring a page reload

**Transparent Operations**
No background activity, no remote connections, no analytics. TypeLess only works when you explicitly interact with it.
